# 🌍 Explorador de Nações

Um catálogo web interativo criado para explorar a diversidade cultural do mundo. Este projeto foi desenvolvido como parte de uma atividade técnica na **Escola Maria Bauab Gibram**.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de páginas individuais por país.
- **CSS3:** Layout moderno utilizando **Flexbox** e **Grid**, com total responsividade para dispositivos móveis.
- **FlagCDN:** Integração de API externa para exibição de bandeiras em alta resolução.

## 📋 Funcionalidades
- Navegação entre catálogo principal e páginas de detalhes.
- Informações sobre Cultura, Gastronomia, Turismo e Curiosidades.
- Design adaptável para diferentes tamanhos de tela.
